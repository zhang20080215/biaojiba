# 标记吧 · 推广内容策略（电影/书单海报墙 + 全平台评分查询）

> 平台：小红书（主）+ 抖音（次）。目标：让人「看到 → 想用 → 搜到小程序 → 截图传播」。
> 本期主推两个功能，下面所有选题、文案、标签都围绕这两个展开。

---

## 一、两个功能的「内容定位」

### A. 电影/书单海报墙
- **本质**：把"我看过的片/书"拼成一张高颜值海报墙 → 天生的**晒图素材**。
- **平台心理**：炫耀欲 + 仪式感 + 年度总结感。小红书爱"晒成果"，抖音爱"刷出整墙的爽感"。
- **核心卖点话术**：
  - "我看过多少部豆瓣Top250？一键生成海报墙，差距一目了然"
  - "微信读书读过的书，拼成一面墙，朋友圈封神"
  - "IMDb Top250 打卡进度可视化，强迫症狂喜"

### B. 全平台评分查询
- **本质**：一个动作查到 **豆瓣 + IMDb + 烂番茄（专业分+观众分）** 四个分 → **效率工具**。
- **平台心理**：实用、解决痛点、"收藏起来下次用"。小红书搜索流量极大（"XX电影值不值得看"）。
- **核心卖点话术**：
  - "选片纠结症必备：一个名字，四个平台评分全出来"
  - "烂番茄专业分和观众分居然差这么多？这片别踩雷"
  - "再也不用开4个App查评分了"

## 二、平台差异化打法

| 维度 | 小红书 | 抖音 |
|---|---|---|
| 流量来源 | **搜索为主**（关键词埋词最关键） | **推荐为主**（前3秒钩子最关键） |
| 内容形式 | 图文封面 + 干货正文 | 竖屏录屏/口播 15-30s |
| 海报墙打法 | 成果展示 + "教你也做一个"教程贴 | 录屏：从空白拼到满墙的"进度爽感" |
| 评分查询打法 | "选片避雷"实用贴 + 搜索词 | "查一部最近热片"现场演示，结果反差做钩子 |
| 转化引导 | 评论区/正文末尾留小程序名 + "搜XX" | 口播引导"小程序搜标记吧"，简介挂名 |

> ⚠️ 平台对站外导流敏感：**正文/图片不要出现"微信""扫码""二维码""外链"，也不要放二维码截图**（易判引流、限流）。统一用中性文字引导"搜『标记吧』小程序"。详见 `合规-推广避坑清单.md`。

## 六、爆款笔记结构模板（小红书图文）

**封面（1张图，决定80%点击）**
- 大字标题（钩子）+ 一张高颜值海报墙截图 / 一张评分对比截图
- 字号要大，手机上一眼看清；用反差数字或"避雷""神器"等情绪词

**正文结构**
1. 钩子开头（痛点或炫耀）：1-2句
2. 干货/演示：3-5步带截图（怎么做 / 怎么查）
3. 结果展示：成图或评分结果
4. 引导：「搜『标记吧』小程序就能用」+ 1句行动召唤
5. 标签：3-6个，搜索词优先

**配图张数**：海报墙类 4-6张（成果图+步骤图）；评分类 3-4张（演示+结果对比）。

## 七、抖音录屏脚本通用骨架（15-30s）

```
0-3s   钩子：直接甩结果（满墙海报 / 四平台评分炸出来）+ 一句话字幕
3-8s   痛点：为什么你需要它（选片纠结 / 看过的片记不住）
8-20s  演示：实际操作录屏，1-2步出结果，节奏要快
20-25s 结果 + 反差：评分差距 / 海报墙满屏
25-30s 引导：「搜『标记吧』小程序」，简介挂名（口播也避免说"微信"）
```
- BGM：选当下热门卡点音乐；关键操作卡在节奏点上。
- 字幕全程在线，静音也能看懂。

## 八、转化路径设计（让流量真的进小程序）

1. **统一记忆点**：所有内容固定一句"搜『标记吧』小程序"，反复强化品牌词（不带"微信"，规避引流风控）。
2. **承接优化（下一步可做）**：被引流来的人第一眼应能立刻上手——可优化分享海报上加品牌名/引导，提升二次截图传播（这部分需改代码，确认方向后我来做）。
3. **数据复盘**：每周记录哪条选题数据最好（点赞/收藏/搜索进店），下周加倍复制该方向。
4. **评论区运营**：高频问题（"评分准吗""怎么做海报墙"）准备好话术，置顶或秒回，评论互动能二次推流。
