# 内容分析 · 全平台电影评分查询配图

## 主题
「标记吧」小程序的全平台电影评分查询功能 —— 一个片名，聚合豆瓣 + IMDb + 烂番茄（Tomatometer 专业分 + Popcornmeter 观众分）。

## 服务的两条笔记
- **笔记4「四项评分怎么看」** —— 评分科普。痛点：单看一个数字容易误判。
- **笔记5「专业分vs观众分科普」** —— 涨知识/反差。痛点：只看一个分会被带偏。

## 内容类型
干货工具种草 + 科普对比。信息密集型（Strategy B）。

## 钩子潜力
- 「选片纠结症有救了」「再也不用开4个App」（痛点共鸣）
- 「影评人吹爆、观众骂烂」（反差，强收藏点）

## 受众
爱看电影、选片纠结、工具党、想避雷的普通观众；小红书搜索「电影评分/选片/烂番茄」的人。

## 互动潜力
- 收藏：高（实用工具 + 科普知识，值得存）
- 评论：中高（「学会扣1」「专业分观众分」易引发讨论）

## 视觉机会
- 4 平台 logo/评分卡片聚合 → 一图看全的「全」感
- 🍅 vs 🍿 双分对比 → 天然的 comparison 布局
- 文艺片 vs 爆米花片的评分反差 → 科普点

## 推荐方案
- 策略：B 信息密集型
- 风格：notion（极简手绘线条、知识卡感，适合评分科普）
- 布局：封面 sparse / 聚合页 balanced / 对比页 comparison / 收尾 sparse
- 语言：中文
- 图片数：5 张

## 图片规划（5张，覆盖两条笔记）
1. **01-cover-评分对比**（笔记4封面，sparse）：同一部电影 · 四项评分怎么看
2. **02-聚合演示**（笔记4内页，balanced/comparison）：输入一个片名 → 豆瓣/IMDb/🍅专业/🍿观众 四张评分卡
3. **03-cover-双分对比**（笔记5封面，sparse/comparison）：影评人吹爆 vs 观众骂烂，差在哪？
4. **04-科普对比**（笔记5内页，comparison）：🍅Tomatometer专业分 vs 🍿Popcornmeter观众分 + 文艺片/爆米花片反差
5. **05-ending-question**（共用收尾，sparse）：这几个评分，你更相信哪个？
