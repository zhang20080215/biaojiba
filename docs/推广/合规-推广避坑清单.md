# 推广合规避坑清单（小红书 / 抖音）

> 目标：所有推广文案 + 图片在发布前，规避平台的**引流、风控、敏感词**机制，降低限流/封号风险。
> 每条内容发布前对照自查。AI 产出推广素材时默认遵守，发现风险词主动替换。

## 一、引流相关（最高优先级）

❌ 禁止出现：
- "微信""WeChat""扫码""二维码""加我""私信""V我""薇""dd"
- 任何二维码截图、外部链接、联系方式（手机号、邮箱、其他平台账号）
- 明显跨平台导流话术（"去XX平台找我""主页有联系方式"）

✅ 替代说法（中性、不带微信）：
- 「搜『标记吧』小程序」
- 「小程序搜标记吧就能用」
- 抖音口播同理：只说"标记吧小程序"，不说"微信"

## 二、绝对化 / 极限词（广告法 + 平台风控）

❌ 避免：最、第一、顶级、唯一、最佳、最好、国家级、史上、永久、100%、绝对、必、根治
✅ 改用：很实用、个人很喜欢、亲测好用、体验下来、对我来说

## 三、诱导互动 / 营销味过重

❌ 避免：免费领、点击进入、限时、抢购、福利、私信领取、关注送、转发抽奖
✅ 改用：真实分享口吻；引导互动用软话术（"你们的进度多少""学会的扣1"）

## 四、夸大 / 虚假承诺

❌ 避免：稳赚、躺赢、一键搞定一切、彻底解决、保证、神器（"神器"可少量用但别堆）
✅ 改用：克制描述真实功能，给出具体场景

## 五、平台调性注意

- 小红书：忌硬广，重"真实体验 + 干货"。带货感太强会限流。
- 抖音：口播别念广告稿；前3秒钩子要自然，结尾引导一句带过。
- 图片：不放二维码、不放微信 logo、不放其他 App 入口截图。

## 六、发布前 30 秒自查

- [ ] 文案/图片里有没有"微信/扫码/二维码/外链/联系方式"？→ 删
- [ ] 有没有"最/第一/免费领/绝对"等极限词、诱导词？→ 换
- [ ] 引导语是不是"搜『标记吧』小程序"这种中性说法？
- [ ] 有没有过度营销、夸大承诺？→ 收敛成真实体验口吻
- [ ] 标签里有没有违禁/导流词？
