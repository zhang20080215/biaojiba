# 标记吧 · 抖音 / 小红书发布合规清单

> 目标：让每条内容在平台内独立成立。品牌只自然露出，不提供跨平台路径，也不使用规避审核的暗语。规则可能变化，发布前以平台最新提示为准。

## 一票否决项

出现任意一项，停止发布并修改：

- 微信、VX、V、威信等平台名称、缩写、谐音或变体。
- 二维码、网址、电话、邮箱、社交账号、外部联系方式。
- “搜索、扫码、加我、私信领取、评论区发你、主页有入口、全网同名”等行动指令。
- 拆字、特殊符号、暗号或分段发送联系方式。
- “最好、第一、全网最全、百分百准确、永久免费”等绝对化或无法证明的承诺。
- “神器、宝藏工具、赶紧去用”等明显营销词堆叠。
- “扣1、关注后发、转发领取”等诱导互动。
- 未获授权的电影片段、影视原声、明星肖像或第三方品牌素材。

## 允许的品牌露出

- 账号名称、头像中的“标记吧”，前提是名称与主体或产品资质一致。
- 真实操作界面顶部自然出现一次产品名。
- 导出海报底部使用小型署名：“标记吧 · 我的专属记录”。
- 正文最多一次客观描述：“这张图用标记吧生成。”不补充获取路径。

## 内容结尾与评论

用内容问题收尾：

- “你看过多少部？”
- “下一期想看哪部电影的评分对比？”
- “专业分和观众分，你更相信哪个？”

用户追问工具时，仅回复：“作品里展示的工具叫标记吧。”不追加平台、入口或联系方式；不要连续复制相同回复。

## 发布前 60 秒检查

- [ ] 标题对用户有独立价值，不是工具广告。
- [ ] 正文、字幕、口播、封面、背景截图没有导流词或联系方式。
- [ ] 产品名自然出现不超过一次，海报署名除外。
- [ ] 没有绝对化承诺、诱导互动和未授权素材。
- [ ] 与另一平台版本在标题、正文或剪辑上有明显差异。
- [ ] 评论预设回复不提供跨平台路径。
- [ ] 发布后 24 小时不频繁修改标题、简介或评论话术。

## 异常处理

遇到仅自己可见、推荐异常或平台警告时，暂停同类表达，不批量删除作品；依次检查封面、字幕、口播、背景、正文、简介和评论。按通知申诉或修改，观察至少 48 小时后再恢复发布。
