# 更新日志

本文件记录「标记吧」小程序各版本的主要变更。最新版本在最上方。

格式约定：每个版本一节，按 **新增 / 优化 / 修复** 分组，只记录用户可感知的变化。

---

## 1.0.28 (2026-07-10)

### 新增
- **「历届金棕榈奖」榜单上线**：戛纳电影节历届金棕榈获奖影片（1955-2025，77 部），含届数 · 导演 · 国家，支持想看 · 已看标记与观影海报；首页置顶卡片带 NEW 标识。
- **「历届奥斯卡最佳原创剧本」榜单上线**：奥斯卡最佳原创剧本历届获奖，编剧功力的年度标杆；首页置顶卡片带 NEW 标识，来自用户许愿（Mi**）。
- **分类页新增「奥斯卡」筛选**：历届奥斯卡最佳影片 / 摄影奖 / 动画长篇 / 原创剧本统一归入奥斯卡分类，与「电影」分开筛选。

### 优化
- **分类页封面视觉统一**：所有主题卡片封面统一叠加主题色调，跟无静态设计图的主题视觉语言一致；NEW 标识改为低饱和度毛玻璃徽标，更精致。
- **灌库匹配更可靠**：新榜单灌库时豆瓣候选改为真实拉取详情校验（原名 / 别名 / 年份），避免同名撞错；灌库前自检脏文本、港台译名、字段缺失，异常直接拦下不写库。
- **全平台电影评分查询**：调整查询超时时长，减少因网络慢导致的查询失败。

## 1.0.27 (2026-07-08)

### 新增
- **三张烂番茄榜单上线**：史上最佳恐怖电影（200 部）、史上最佳战争电影（150 部）、史上最佳动画电影（138 部），数据取自烂番茄权威榜单，配豆瓣简体片名 / 评分 / 封面；支持想看 · 已看标记与观影海报（海报墙 + 文字卡片），首页各带独立主题色卡片（暗红 / 钢蓝 / 橙）与 NEW 标识。

### 优化
- **通用榜单数据实时刷新**：走共享集合的榜单（含奥斯卡最佳摄影及本次三张新榜）进入列表页即拉取最新数据，避免榜单更新后被 24 小时本地缓存挡住。

## 1.0.26 (2026-07-04)

### 修复
- **iOS 封面空白**：奥斯卡最佳摄影 / 最佳动画长篇、每日电影、全平台评分查询等卡片封面在 iOS 上不显示（webp 格式在 iOS 真机无法渲染），改用 jpg 后恢复正常。
- **奥斯卡最佳摄影 · 文字海报排版**：年份与片名在电影较多时会叠在一起 —— 改为字号随格子自适应、垂直居中堆叠，不再重叠。
- **奥斯卡最佳摄影 · 海报墙封面变形**：导出的封面被拉伸压扁 —— 改为按原图比例填充（2:3），不再变形。
- **奥斯卡最佳摄影 · 列表按钮**：已看 / 想看按钮文字未水平居中，已修正。

### 优化
- **奥斯卡最佳摄影 · 海报更清晰**：导出尺寸提升至 1242×1660；文字海报每行展示更多、字号更大。

## 1.0.24 (2026-07-02)

### 新增
- **「历届奥斯卡最佳摄影奖」榜单上线**：历届最佳摄影获奖作品（大陆标准简体片名），支持想看 / 已看标记与海报墙分享；首页置顶并带 NEW 标识。
- **「许愿新榜单」需求收集**：首页分类标签下方新增许愿入口，可提交想要的片单 / 书单（每人每天 5 条），采纳后上线。
- **许愿来源标识**：奥斯卡最佳动画长篇、最佳摄影奖卡片封面标注许愿用户（安然**、And**）。

### 优化
- **启动提速**：广告配置、用户身份获取等非关键任务延后到首屏渲染后；主题色 / 用户信息改为内存缓存，消除启动期反复的同步存储读写（首页 onLoad 长任务与同步 API 告警清零）。
- **首页封面零闪烁**：奥斯卡动画 / 摄影卡片封面改为内置静态图（Flow《猫猫的奇幻漂流》/《粗野派》海报），不再有"默认图 → 真封面"的切换。
- **包体瘦身**：删除废弃空页面（mark / stats / generate），超尺寸封面图降采样转 webp，杂项文档移出代码包。

### 修复
- **兜底海报裂图**：`default-movie.jpg` 实为损坏文件（误存的文本），导致每日电影 / 读书候选列表、票房榜等 20+ 处兜底海报显示裂图，已替换为正常占位图。
- **许愿提交成功提示被截断**：带图标 toast 超 7 字符会截断，文案已调整。

## 1.0.23 (2026-06-23)

### 新增
- **「每日运动」主题上线**：按月记录每次训练，支持日历 / 时间轴视图。动作分有氧 / 力量 / 拉伸·柔韧三大类，力量按身体部位（胸 / 背 / 腿 / 肩 / 手臂 / 臀 / 核心）分组，40+ 项；可一次添加多组动作，记录支持拖拽排序、左滑编辑 / 删除。
- **运动卡片分享**：选中某天可生成运动打卡卡片（小红书 3:4），内置经典蓝 / 薄荷绿 / 樱花粉 / 暖橙 4 套主题色，自动记住上次选择。
- **「奥斯卡最佳动画长篇」榜单上线**：历届最佳动画长篇获奖作品，支持想看 / 已看标记与海报墙 / 文字海报分享；首页置顶并带 NEW 标识，封面取榜单首部影片。

### 优化
- **「每日推荐电影」每天自动更换**：从豆瓣评分 8.5 分以上、已收录的电影中按天轮换推荐。
- **最近查询记录按最新查询时间排序**：再次查询同一部电影会置顶。
- **奥斯卡动画分享海报**：电影墙每行数量随片数自适应（海报不再被拉长），文字海报减少留白、增大字号，统一为 1080×1440 小红书比例。

### 修复
- **电影评分自动刷新**：查询过的电影数据超过 24 小时后再次查看时，自动重新抓取豆瓣 / IMDb / 烂番茄最新评分。

## 1.0.22 (2026-06-17)

### 新增
- **「每日读书」主题上线**：按月记录读过的书，支持日历、书墙、时间轴三种视图。
- **「时间轴」视图**：每日电影、每日读书新增「时间轴」标签，按日期展示当月观影 / 阅读流水（封面 + 评分 + 心情 + 短评）。
- **记录左滑编辑**：观影 / 阅读记录卡片左滑可「编辑 / 删除」，编辑通过底部弹窗快速修改 评分 / 心情 / 短评。

### 优化
- **评分改为五角星**：支持拖拽点亮，半颗星精度，最少半颗。
- **心情选择「更多」**：常用心情之外，点「更多」可在弹窗中选择更多心情。
- **电影搜索升级**：改用豆瓣搜索结果页，搜索结果直接显示豆瓣评分 / 评价人数，并支持模糊匹配（如「盗梦」可出「盗梦空间」）。
- **添加记录按钮**：改为底部吸底操作栏，文案更明确（添加观影记录 / 添加阅读记录）。
- **首页**：新增「每日标记 / 主题标记」分区标题；顶部 Hero 区高度收紧。

### 修复
- 修复记录卡片左滑时，操作按钮与卡片之间的圆角割裂。
