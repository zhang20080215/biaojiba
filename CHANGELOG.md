# 更新日志

本文件记录「标记吧」小程序各版本的主要变更。最新版本在最上方。

格式约定：每个版本一节，按 **新增 / 优化 / 修复** 分组，只记录用户可感知的变化。

---

## 1.0.23 (2026-06-23)

### 新增
- **「每日运动」主题上线**：按月记录每次训练，支持日历 / 时间轴视图。动作分有氧 / 力量 / 拉伸·柔韧三大类，力量按身体部位（胸 / 背 / 腿 / 肩 / 手臂 / 臀 / 核心）分组，40+ 项；可一次添加多组动作，记录支持拖拽排序、左滑编辑 / 删除。
- **运动卡片分享**：选中某天可生成运动打卡卡片（小红书 3:4），内置经典蓝 / 薄荷绿 / 樱花粉 / 暖橙 4 套主题色，自动记住上次选择。
- **「奥斯卡最佳动画长篇」榜单上线**：历届最佳动画长篇获奖作品，支持想看 / 已看标记与海报墙 / 文字海报分享；首页置顶并带 NEW 标识，封面取榜单首部影片。

### 优化
- **「每日推荐电影」每天自动更换**：从豆瓣评分 8.5 分以上、已收录的电影中按天轮换推荐。
- **最近查询记录按最新查询时间排序**：再次查询同一部电影会置顶。
- **奥斯卡动画分享海报**：电影墙每行数量随片数自适应（海报不再被拉长），文字海报减少留白、增大字号，统一为 1080×1440 小红书比例。

### 修复
- **电影评分自动刷新**：查询过的电影数据超过 24 小时后再次查看时，自动重新抓取豆瓣 / IMDb / 烂番茄最新评分。

## 1.0.22 (2026-06-17)

### 新增
- **「每日读书」主题上线**：按月记录读过的书，支持日历、书墙、时间轴三种视图。
- **「时间轴」视图**：每日电影、每日读书新增「时间轴」标签，按日期展示当月观影 / 阅读流水（封面 + 评分 + 心情 + 短评）。
- **记录左滑编辑**：观影 / 阅读记录卡片左滑可「编辑 / 删除」，编辑通过底部弹窗快速修改 评分 / 心情 / 短评。

### 优化
- **评分改为五角星**：支持拖拽点亮，半颗星精度，最少半颗。
- **心情选择「更多」**：常用心情之外，点「更多」可在弹窗中选择更多心情。
- **电影搜索升级**：改用豆瓣搜索结果页，搜索结果直接显示豆瓣评分 / 评价人数，并支持模糊匹配（如「盗梦」可出「盗梦空间」）。
- **添加记录按钮**：改为底部吸底操作栏，文案更明确（添加观影记录 / 添加阅读记录）。
- **首页**：新增「每日标记 / 主题标记」分区标题；顶部 Hero 区高度收紧。

### 修复
- 修复记录卡片左滑时，操作按钮与卡片之间的圆角割裂。
