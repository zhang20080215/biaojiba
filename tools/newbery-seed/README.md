# 纽伯瑞儿童文学金奖 灌库说明（newbery）

走书籍通用流水线（`enrichThemeBooks` → `generic_theme_books` → `getThemeBooks`），前端共享 `pages/genericBookList`，主题配置在 `utils/genericBookThemeConfig.js`（`newbery` 一行）。

数据源：维基百科「纽伯瑞奖」词条。**范围：仅「金奖」(Newbery Medal)，且仅收录维基有中文译名的，共 77 部（1922–2022）**。无中文版的约 34 部（多为 1920–50 年代老书 + 2023–2025 最新三部）已剔除——外文书靠中文书名匹配豆瓣，无中文版的查无、缺封面。

字段：`rank`（新到旧连续编号，rank 1=2022 最新，配合 `orderDirection:'asc'`）、`year`（获奖年）、`title`（中文译名）、`author`（作者）。**无 `edition` 字段**——纽伯瑞按年评选无「届」概念，配置用 `showYear:true`（列表只显示「年份 · 作者」，不显示「第X届」）。

## 灌库

`enrichThemeBooks` 云端测试，粘 `newbery.params.json` 整份。`autoContinue:true` 跑完自动接力，只需点一次。灌完 `getThemeBooks {theme:"newbery"}` 应返回 77 条。

## ⚠️ 匹配质量：外文书 + 港台译名，自动命中率偏低

跟茅盾（纯中文书、豆瓣一定有）不同，这批是**外文童书**，且维基给的部分是**港台译名**，跟豆瓣大陆译名常对不上（书籍流水线按规范化书名精确匹配 + subject_suggest 兜底）。预计相当一部分需人工核对：

- **易失配示例**：《时间的皱折》《记忆传授人》《大嗓门传奇》《喜马拉雅的传信鸽》(豆瓣或作《花颈鸽》)、《好心的大爷！帮帮忙！…》(超长题名) 等。
- 灌完看日志 `[未验证匹配]` / `匹配未通过校验` 告警，逐条去豆瓣核实，在对应条目加 `"doubanId": "xxx"` 重灌（同 palmeDor《樱桃的滋味》处理）。
- 有 11 条作者维基无中文名，暂填英文原名（`author` 仅作展示 + 匹配二次确认，非阻断项）：rank 8/30/47/49/58/60/62/64/66/71/73。核实豆瓣后可补中文译名。

建议：先整份灌一遍拿到命中情况，再据告警集中补 `doubanId` / 修订书名跑第二遍。
